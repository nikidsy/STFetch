# STRANGER THINGS - API

O STFetch é uma aplicação web que permite pesquisar personagens da série **Stranger Things** de forma rápida e interativa.
O projeto ultiliza **JavaScript e Fetch API** para consumir dados de uma API externa e exibir informações sobre os
personagens diretamente na página.

Ao digitar o nome de um personagem no campo de busca, o site realiza uma requisição assíncrona para a API, recebe os dados
em formato **JSON** e apresenta infomações como **nome, imagem e curiosidades sobre o personagem.**

Este projeto foi desenvolvido co o objetivo de praticar **consumo de APIs, JavaScript assíncrono e manipulação do DOM**,
além de reforçar conceitos fundamentais do desenvolvimento web. Esse tipo de informação é chamada de data fetching, 
ou seja, buscar dados de uma API e mostrar no site dinamicamente.

## FUNCIONALIDADES
- *Buscar personagens da série Stranger Things.*
- *Exibir informações e imagens dos personagens*
- *Carregamento de dados de forma dinâmica*
- *Interface simples e interativa*


## TECNOLOGIAS UTILIZADAS
- **HTML**: Estrutura da página.
- **CSS**: Estilização.
- **JavaScript**: Lógica do site.
- **Fetch API**: Comunicação com API
- **API PÚBLICAS DE SÉRIES(TVMAZE)**: Dados sobre personagens de Stranger Things.
